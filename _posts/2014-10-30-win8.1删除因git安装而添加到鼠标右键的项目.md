## 删除右键菜单中的 git相关栏目

**windows x64**

cmd进入"D:\Program Files (x86)\Git\git-cheetah"目录，运行

```
regsvr32 /u git_shell_ext64.dll
```

**windows x32**

cmd进入"D:\Program Files\Git\git-cheetah"目录，运行

```
regsvr32 /u git_shell_ext.dll
```

可注销右键菜单.
